Steps To Create a Successful Campaign on Zsale Launchpad

These are the steps to create a Successful Campaign

Before creating any campaign of your choice you must have the following requirement:
  1. Make sure you have enough funds in your wallet
  2. Make sure you have the Supported Network Ids in your wallet Chain Id are as Follows
    • Ethereum Chain Id (Mainnet)
    • Smart Chain Id (Mainnet and Testnet)
    • Meter Chain Id (Mainnet and Testnet)
    • Hardhat Chain Id (Testnet)
    • Aurt Chain Id (Mainnet)
  3. Make Sure You are Logged in to your wallet and connect to the site with your wallet
